# Scientific Calculator

A responsive scientific calculator built with HTML, CSS, and JavaScript.

### Features
- Basic operations: +, -, ×, ÷, %
- Scientific functions: sin, cos, tan, log, ln, √, x^y, π, e
- Memory functions: M+, MR, MC
- Degree / Radian toggle
- Dark / Light theme toggle
- Calculation history

### Live Demo
[Click here to use the calculator](https://nqobile2004.github.io/scientific-calculator/)

### How to use
Just open the live link above or download `index.html` and open it in your browser.
